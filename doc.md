# Emma


### Stack used

- node express
- knex + bookshelf mysql
- vanilla JS / jQuery frontends

### Installation

Emma requires [Node.js](https://nodejs.org/) v6+.

Install the dependencies and start the server.

Development
```sh
$ cd emma
$ npm install
$ npm run start:dev
```

For production environments...
heroku runs npm run start
